### A. Google Search Console
```bash
1. Verificar propiedad en Google Search Console
   - Agregar NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION en .env
   - Verificar sitemap.xml
   - Enviar todas las URLs

2. Monitorear:
   - Core Web Vitals
   - Errores de rastreo
   - Cobertura de índice
   - Experiencia de página
```

### B. Google Analytics 4
```bash
1. Crear cuenta GA4
2. Agregar NEXT_PUBLIC_GOOGLE_ANALYTICS en .env
3. Implementar eventos personalizados:
   - Clics en "Contactar"
   - Envío de formularios
   - Descargas de portafolio
   - Tiempo en página
```

### C. Google Tag Manager
```bash
1. Crear cuenta GTM
2. Agregar NEXT_PUBLIC_GOOGLE_TAG_MANAGER en .env
3. Configurar tags:
   - Conversiones
   - Remarketing
   - Eventos personalizados
```

### D. Velocidad y Core Web Vitals
```bash
# Objetivos:
- LCP (Largest Contentful Paint): < 2.5s ✓
- FID (First Input Delay): < 100ms ✓
- CLS (Cumulative Layout Shift): < 0.1 ✓
- FCP (First Contentful Paint): < 1.8s ✓
- TTI (Time to Interactive): < 3.8s ✓

# Herramientas de medición:
- PageSpeed Insights
- Lighthouse
- WebPageTest
- GTmetrix
```
